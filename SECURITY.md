# Security Policy

## Supported Versions

The table below outlines the security support status for the Nexa Platform:

| Version | Supported | Notes |
|---|---|---|
| 1.0.0 | :white_check_mark: Yes | Current Stable Production Release |
| < 1.0.0 | :x: No | Pre-release and development versions |

## Reporting a Vulnerability

If you discover a security vulnerability within the Nexa Platform, please do not open a public issue. Instead, report it privately.

You can report vulnerabilities by:
1. Sending an email to the project maintainers: **diegosandoval284@gmail.com**.
2. Using the **Private Vulnerability Reporting** feature on GitHub (available on the repository's Security tab).

We will acknowledge receipt of your report within 48 hours and provide a detailed response with next steps within 7 days.
