# Nexa Platform v0.6.0 - Advanced Context Integration & DB Schema Expansion

## Overview
This release expands the C# backend API to support a total of 73 operational endpoints across our core Bounded Contexts. It introduces database schema migrations, seed scripts, and full endpoint integration to act as a single source of truth for the B2B web application.

## Main Changes

### Logistics Context
- **Shipments Management:** Added `Shipment.cs` aggregate, `DeliveryStatus` value objects, and custom `ShipmentsController` to manage dispatch tracking and order timeline events.

### Invoicing Context
- **Payments & Billing:** Fully implemented `Payment.cs` and `Invoice.cs` aggregates, including cancellation, registration, and update commands.

### Warehouse Context
- **Inventory Control:** Expanded stock reservation, low stock queries, and item reservation logic.

### Database Migrations & Seeds
- **EF Core Migrations:** Created and applied the `AddAv2Batch2Resources` migration containing Category, Brand, Order, Payment, and Warehouse schemas.
- **Local Database Seeds:** Authored `seed-av2-local.sql` to preload the database with 58 catalog items, 10 warehouses, 15 orders, and commercial transactions ready for WebApp consumption.
