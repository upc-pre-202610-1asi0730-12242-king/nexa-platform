# Nexa Platform - Pre-Release v0.7.0

This release represents **Version 0.7.0** of the **Nexa Backend Platform**, prepared by **Gerard Rojas** (`GerardRojasMancilla <u202413142@upc.edu.pe>`).

This version concentrates on the core database migration from MySQL to PostgreSQL (Npgsql) and setting up initial database migrations.

---

## Key Changes in v0.7.0
- Migrated database provider from MySQL to PostgreSQL using `Npgsql.EntityFrameworkCore.PostgreSQL`.
- Automatically configured snake_case table and column conventions in `AppDbContext`.
- Generated `InitialCreate` database migrations for PostgreSQL.
- Added local SQL database seed script `seed-local.sql` containing 58 catalog items, categories, brands, and warehouses.
- Configured appsettings placeholders to isolate database parameters.
